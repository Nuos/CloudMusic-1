## CloudMusic made by Qt

 这个小demo是模仿网易云音乐的界面做的音乐播放器，开发环境是ubuntu18+Qt5；

起因是：网易云音乐播放器在ubuntu18上有莫名奇妙的bug 只适配到ubuntu16，于是想着自己动手
